How to Use a Biography for your Project Comments
To use a biography in your project comments you must first identify an existing biography that you want to use or create a new. At the listing of authors, try to find an existing record for the author. If you don't find one, create a new record. For each author there can be any number of biographies — you can either decide to use any of the ones that may exist or enter a new biography.

Each biography has its own number which can be used in a special markup like this: [biography=13]. All you need to do to include the biography is put this in your comments:

[biography=13]

You may also include other comments in addition to the biography as well as more than one biography. For example,

For the interested, there's a biography of the author:
[biography=13]
This edition was edited by Humphrey, presented below:
[biography=29]
